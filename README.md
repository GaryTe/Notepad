# Notepad ( в процессе создания )

## Запуск программы
- Скачайте и установите интерпретатор Python ( страница загрузки python.org/downloads )
- Откройте интерпретатор Python и через меню File -> Open откройте файл main.py
- Запустите файл main.py через меню Run -> Run Module, или нажмите клавишу F5
